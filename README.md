```
<User>            [Core Smart Contract]           [Price Oracle]              [Exchange Privider]       <Exchange>
  |------- send eth ------->|
                            |------- request price ----->|
                                                         |------------------- request price --------------->|
                                                         |<------------------ response price ---------------|
                            |<------ response price -----|
                            |------- split orders ------- && ------ make orders ------->|
                                                                                        |---- exchange ---->|
                                                                                        |<--- done ---------|
  |<----------------------- orders completion && return ask tokens ---------------------|
  ```
